# 🃏 Balatro Project

Site web dédié au jeu vidéo **Balatro**, un roguelike de cartes solo développé par LocalThunk. Ce projet présente le jeu, ses règles, sa communauté de modding et du contenu multimédia.
Le but est de présenter le jeu afin de dévellopper la communauté française peu présente sur le jeu.
## 📄 Pages

| Page | Contenu |
|------|---------|
| **Accueil** | Présentation générale de Balatro — concept, fonctionnalités clés, style roguelike |
| **Règles** | Grille explicative des mécaniques de jeu : mains de poker, jokers, blinds, antes, boutique |
| **Modding** | Galerie de mods communautaires avec carrousel d'images et descriptions |
| **BMP** | Vidéo de présentation d'un concept multijoueur pour Balatro |
| **Contact** | Formulaire de contact sobre avec champs pseudonyme, email et adresse |
## ✨ Fonctionnalités

- 🌍 **Bilingue EN / FR**
- 📱 **Responsive** — adapté aux écrans mobiles et tablettes

## 🚀 Déploiement
Le site est hébergé sur **Vercel**, connecté automatiquement à ce repo GitHub.
